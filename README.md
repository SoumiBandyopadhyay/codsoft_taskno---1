# 📝 To-Do List Application

A simple and user-friendly **To-Do List Application** built using **Python**.
This project helps users organize and manage their daily tasks efficiently through a command-line interface (CLI).

---

## 📌 Features

* ➕ Add new tasks
* 📋 View all tasks
* ✏️ Update existing tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 💾 Save tasks permanently using JSON file storage
* 🔄 Automatically load saved tasks when the program starts
* 🖥️ Easy-to-use command-line interface

---

## 🛠️ Technologies Used

* Python 3
* JSON (for data storage)
